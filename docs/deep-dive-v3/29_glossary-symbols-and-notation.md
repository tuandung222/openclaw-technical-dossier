# 29 - Glossary Symbols And Notation

## Symbols

- `X_t`: global runtime state at time `t`
- `R`: routing function
- `Cap_eff`: effective capability set
- `L_total`: end-to-end latency decomposition

## Terms

- Composition root: module wiring dependencies at startup
- Policy lattice: ordered hierarchy of policy sources
- Capability calculus: set-theoretic view of executable tools
- Eventual consistency: async convergence model for indexed memory state
- Identifiability: khả năng truy vết nguyên nhân từ observed signals

## Module aliases used in docs

- gateway: `src/gateway/*`
- agent runtime: `src/agents/*`
- memory: `src/memory/*`
- plugins: `src/plugins/*`, `src/channels/plugins/*`
- security: `src/security/*`
