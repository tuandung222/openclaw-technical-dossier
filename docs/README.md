# Documentation Layout

- `docs/baseline-v1/`: bộ tài liệu phiên bản đầu tiên (đã giữ nguyên nội dung cũ)
- `docs/deep-dive-v2/`: bộ super deep-dive cập nhật theo snapshot OpenClaw mới nhất
- `docs/deep-dive-v3/`: bộ ultra deep-dive (PhD-level) với formal model + Mermaid + call-flow theo file/function

Nếu bạn muốn đọc bộ mới nhất, bắt đầu từ:

- `docs/deep-dive-v3/00_navigator.md`
